# Origami samples
</div>

Примеры демонстрируют использование Origami Framework в различных сценариях взаимодействия. 
Проекты выполняются с помощью Maven и используют Java 17.

## Примеры

Раздел примеров Origami содержит примеры в виде небольших проектов, каждый из которых покрывает несколько тестовых случаев.

Ниже представлен список примеров и их основные цели

| Пример                               |                                Описание                                 |
|--------------------------------------|:-----------------------------------------------------------------------:|
| [sample-kafka](sample-kafka)         |                  Демонстрирует взаимодействие с Kafka                   |
| [sample-hibernate](sample-hibernate) |   Демонстрирует взаимодействие с базами данных(подключение, запросы)    |
| [sample-rest](sample-rest)           |            Демонстрирует вызовы HTTP REST в качестве клиента            |
| [sample-websocket](sample-websocket) |        Демонстрирует подключение к Websocket в качестве клиента         |
| sample-ibm-mq (TODO)                 |                  Демонстрирует взаимодействие с IBM MQ                  |
| sample-selenide (TODO)               |                Демонстрация тестирования WEB интерфейса                 |
| sample-test-containers               | Демонстрация сборки test containers. Примеры приведены в каждом проекте |
